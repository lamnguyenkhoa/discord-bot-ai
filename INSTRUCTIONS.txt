HOW TO RUN THE DISCORD BOT
==========================

1. FIRST TIME SETUP
-------------------
a) Copy the env template:
   cp .env.example .env

b) Edit .env and fill in your credentials:
   DISCORD_TOKEN=your_discord_bot_token
   OPENROUTER_API_KEY=your_openrouter_api_key
   MODEL_NAME=openai/gpt-4o-mini   (change if desired)
   SUMMARIZE_THRESHOLD=50           (summarize log after this many exchanges)

c) Install Python dependencies:
   pip install -r requirements.txt

d) In Discord Developer Portal (https://discord.com/developers/applications):
   - Go to your bot -> Bot tab
   - Enable "MESSAGE CONTENT INTENT" under Privileged Gateway Intents
   - Save changes

2. RUN THE BOT
--------------
   python bot.py

   You should see: "Logged in as <BotName> (ID: ...)" in the console.
   The bot is now online. Keep this terminal open.

3. USING THE BOT
----------------
   - @mention the bot in any channel to get a reply
   - The bot ignores all messages that don't @mention it
   - Conversation logs are saved to memory/YYYY-MM-DD.md

4. STOP THE BOT
---------------
   Press Ctrl+C in the terminal.

5. NOTES
--------
   - memory/ folder is auto-created on first run
   - memory/ is gitignored (logs stay local)
   - If you change MODEL_NAME in .env, restart the bot for it to take effect
   - If the bot replies "Sorry, I'm having trouble thinking right now."
     check that OPENROUTER_API_KEY is valid and has credits
